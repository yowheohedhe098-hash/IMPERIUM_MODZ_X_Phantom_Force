# IMPERIUM MODZ X Phantom Force - Análise do Projeto

## Estrutura Atual
- **Tipo**: Projeto Android NDK com JNI (C++)
- **SDK**: Android 29-30
- **ABIs Suportadas**: armeabi-v7a, arm64-v8a, x86
- **Versão**: 3.2

## Componentes Principais
1. **KhanhMods** - Framework de modificação
   - IL2CppSDK - Integração com IL2Cpp
   - Tools - Utilitários diversos
   - Dobby - Hook framework
   
2. **Bibliotecas Externas**
   - libcurl - Requisições HTTP
   - OpenSSL - Criptografia
   - Munodz - Memory search

3. **Estrutura JNI**
   - ImGui - Interface gráfica
   - Memory patching
   - Shared preferences
